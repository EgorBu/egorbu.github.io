Discover - rovers

  • Retrieve repository URLs from multiple repository hosting providers

Clone - gitcollector

  • Download and update git repositories at large scale
  • Data efficient storage format siva

Classify - enry

  • Programming language detector and toolbox to ignore binary or vendored files
  • ~10x faster than linguist

Parsing - bblfsh

  • Extendable - there are 400+ PL in the world
  • Languages - ~10 supported
  • Scalable - design with idea of processing all source code in the world
  • Extract AST in universal format
    • Universal API - no boiler plate for processing different languages
    • Universal roles - unification of roles among different languages

$$ V_i \cdot V_j = PMI_{ij} = \log\frac{C_{ij} \sum C}{\sum_{k = 1}^N C_{ik}\sum_{k = 1}^N C_{jk}} $$

Machine Learning

Rules

  • a≤5 Λ b≤1 Λ c ⇒ α
  • a≤5 Λ 1<b<4 ⇒ β
  • 5<a<10 Λ c ⇒ γ
  • a>5 Λ c Λ b>2 ⇒ α

Rules optimization

a>5 Λ c Λ b>2 Λ d Λ a>10 ⇒ α(merge)

a>10 Λ c Λ b>2 Λ d ⇒ α(redundant)

a>10 Λ c Λ d ⇒ α(feature exclusion)

a>10 Λ c ⇒ α(confidence threshold)

Result

  • -90% rules with no quality loss
